Filer

Ditt filområde är en förvaringsplats för mappar och filer, som kan användas i vyer.

Antalet filer du kan ladda upp på ditt filområde beror på diskutrymmet, som beviljats dig. Diskutrymmets kvot syns till höger.

Du kan dra och släppa för att ordna dina filer och mappar.

Av säkerhetsskäl kan tjänst administratörerna begränsa uppladdningen av vissa filtyper.
