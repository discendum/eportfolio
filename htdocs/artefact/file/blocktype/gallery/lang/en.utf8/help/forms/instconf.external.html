External gallery URL

Insert external gallery URL to the text box.

It is possible to embed the following external galleries:

  • Flickr, e.g. https://www.flickr.com/photos/12345678@N01/sets/98765432123456789/
  • Picasa, e.g. http://picasaweb.google.com/user.name/AlbumName#
  • Panoramio, e.g. http://www.panoramio.com/user/123456
  • Photobucket, e.g. http://s9.photobucket.com/user/username/library or http://s9.photobucket.com/user/username/library/albumname
