Fält som kan användas vid sökning

Profilfält som är markerade med "fält som kan användas vid sökning" är tillgängliga för alla inloggade användare.

Förnamn, efternamn och alias är oftast alltid fält som kan sökas i . Namnsökningen kontrolleras ändå av tjänstinställningarna "Användare kan dölja namn" och "Sök efter användarnamn", och även av användarinställningar.

Andra fält som kan användas vid sökning, såsom e-postadress kan användas vid sökning, men de visas inte i sökresultaten.
