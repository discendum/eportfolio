Redigera sida

Genom att planera layouten på din sida kan du personalisera sidan med dina resurser och din information. Om du till exempel bara vill visa ett blogginlägg, klicka på bloggfliken och dra sedan blocket för blogginlägg till din sida. Kom ihåg att klicka på Spara när du lägger till nya element på din sida.

När du bygger upp innehållet på din layout kan du kolla hur den kommer att se ut genom att klicka på 'Visa min sida'. Klicka på 'Tillbaka' för att fortsätta redigera.
