Sök efter vänner

Här kan du söka efter användare, som du kan lägga till som vänner. Beroende på deras vänkontroll kan du automatiskt läggas till som deras vän eller så kan du skicka en vänförfrågan. Det kan hända, att några användare inte tillåter vänförfrågan alls.

Då du skickar en vänförfrågan kan du skriva ett kort meddelande, där du introducerar dig själv. Det här är viktigt speciellt om du använder ett alias då andra inte ser ditt riktiga namn.
