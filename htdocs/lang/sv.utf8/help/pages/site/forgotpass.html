Användarnamns-/Lösenordspåminnare

Om du har glömt ditt användarnamn eller lösenord kan webbtjänsten skicka dig ett meddelande med ditt användarnamn. Genom att klicka på länken i meddelandet kan du nollställa ditt lösenord. Skriv in ditt användarnamn eller din primära e-postadress i det givna fältet och klicka på "Skicka förfrågan". Du får automatiskt ett e-postmeddelande. För att ändra på lösenordet, klicka på länken i e-postmeddelandet för att bekräfta nollställningsförfrågan.

Varning: Ifall du får ett e-postmeddelande om lösenordsändring utan att du bett om det, kan det vara en förfalskning. Klicka inte på några länkar i ett sådant e-postmeddelande.
