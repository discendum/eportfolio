Offentliga grupprum

Ifall du vill ge alla användare rättigheten att skapa grupprum, som även andra än webbtjänstens medlemmar kan se, välj 'Alla'.

I ett offentligt grupprum kan vem som helst, som hittar sidan (såsom sökmotorer) se grupprum, sidan "Om grupprummet", listan på medlemmar och alla diskussioner i forumen. Man måste ändå registrera sig på webbtjänsten innan man t.ex. kan lägga in något på forumen eller se på användarnas profiler.

För att granska listan av grupprumsmedlemmar kan begränsas för både medlemmar och icke-medlemmar i grupprummet av grupprums administratorn.
