Standard licens

Använd denna inställning för att ange standard licens för innehållet som dina användare skapar eller laddar upp på webbtjänsten.

Användare kan överköra den här på deras "Inställningar" sida och på varje element som de skapar eller laddar upp.

Obs: Om du inte är säker på vilken standard licens att välja, vänligen konsultera organisationens jurist eller en copyright jurist.
