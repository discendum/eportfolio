Felmeddelande vid inloggning

Då värdbefogenheter inte existerar, kan användarna endast logga in via SSO. De kan inte använda Maharas inloggningsformulär.

Skriv här in meddelandet som visas för användarna, då de försöker logga in via Maharas inloggningsformulär. Meddelandet borde innehålla instruktioner över, hur användaren kan logga in på Mahara via SSO.
