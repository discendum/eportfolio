Fullmaktsgivarvärd

Ifall du definierar en fullmaktsgivarvärd, kan användare logga in genom att använda den ifrågavarande fullmakten, men även SSO.

Du kan t.ex. definiera LDAP-autentisering och att den ska vara den här fullmaktens värd. Det betyder, att användare kan logga in genom Maharas inloggningsformulär, som använder LDAP-personinformation. Användare kan logga in via SSO, genom att använda en utomstående fullmakt.

Du behöver inte definiera en fullmaktsgivarvärd. Ifall du inte gör det, kan användare som använder den här fullmakten logga in via Mahara endast genom SSO.
