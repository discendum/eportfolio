Läroanstaltens kontroll värde

Det här är ett värde, som observeras mot Institutionens egenskapsvärde, som Identity Provider (IdP) hoppat över. Den visar vilken läroanstalter användarna hör till. Om Institutionens Regex är vald, kan det här värdet vara ett reguljärt uttryck, som kan användas vid kontrollering mot läroanstaltens egenskapsvärde.
