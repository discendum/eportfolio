Miten upotan?

Google Docs

Google Apps for Educational domains: Yksittäisten tiedostojen jakaminen toimii samalla tavalla, kuin tavallisilla Google Apps -tileillä.

Oppilaitoksen ylläpitäjä on saattanut rajoittaa jaetut tiedostot näkymään vain oppilaitoksen jäsenille.
Tällöin näytetään Google Apps -kirjautumisikkuna ja upotetut tiedostot näytetään kirjautumisen jälkeen.

Documentit

  1. Avaa Google Docs -dokumentti
  2. Klikkaa Share ja Publish to the Web (oikeassa ylänurkassa)
  3. Klikkaa Start publishing -nappia
  4. Kopioi Document link tai Embed code
  5. Liitä GoogleApps-lohkoon

Taulukot

  1. Avaa Google Docs -taulukko
  2. Klikkaa Share ja Publish as a web page (oikeassa ylänurkassa)
  3. Klikkaa Start publishing -nappia
  4. Kopioi linkki
  5. Liitä GoogleApps-lohkoon

Esitykset

  1. Avaa Google Docs -esitys
  2. Klikkaa Share ja Publish / embed (oikeassa ylänurkassa)
  3. Klikkaa Publish document -nappia
  4. Kopioi linkki
  5. Liitä GoogleApps-lohkoon

Kokoelmat

  1. Google Docsissa klikkaa My collections tai Collections shared with me (lista vasemmalla)
  2. Klikkaa ja valitse kokoelma, jonka haluat upottaa
  3. Klikkaa Sharing - Settings -linkkiä (oikealla puolella)
  4. Vaihda oikeudet julkiseksi ja tallenna muutokset
  5. Kopioi linkki
  6. Liitä GoogleApps-lohkoon

Google Docs Viewer

Tämä toiminto on samanlainen kuin Gmailin esikatselu. Voit upottaa tiedostoja, jotka ovat suoraan julkisia Internetissä. Kopioi ja liitä tiedoston koko osoite (tiedostonimi mukaanlukien) GoogleApps-lohkoon.

Google Docs Viewer tukee seuraavia tiedostotyyppejä: PDF, TIFF, PPT, DOC, DOCX.

Huomio: Maharaan ladatut tiedostot eivät ole suoraan julkisia, joten niitä ei voi upottaa.

Google Calendar

  1. Avaa Google Calendar
  2. Klikkaa Settings ja Calendar settings (valikossa oikealla ylhäällä)
  3. Klikkaa Calendars -linkkiä vasemmalla ylhäällä
  4. Klikkaa upotettavan kalenterin nimeä
  5. Calendar details -välilehdellä on toiminto Embed This Calendar
  6. Muokkaa kalenterin ulkoasua ja kopioi upotuskoodi
  7. Liitä GoogleApps-lohkoon

Google Maps

  1. Avaa Google Maps
  2. Klikkaa My Maps -linkkiä vasemmalla ylhäällä
  3. Klikkaa upotettavan kartan nimeä
  4. Klikkaa Link -linkkiä oikealla ylhäällä
  5. Muokkaa kartan ulkoasua ja kopioi upotuskoodi
  6. Liitä GoogleApps-lohkoon
Huomio: Voit ohittaa kohdat kaksi ja kolme, jos upotat muita kuin omia karttojasi.

Google Books

  1. Avaa Google Books
  2. Hae kirja, jonka haluat upottaa
  3. Klikkaa Preview this book -linkkiä tai nappia
  4. Klikkaa Link -linkkiä oikealla ylhäällä
  5. Kopioi kirjan upotuskoodi
  6. Liitä GoogleApps-lohkoon
