Hur bäddar man in?

Google Docs

Läroanstaltens admin kan ha gjort en begränsning så att delade filer visas endast för läroanstaltens medlemmar.

Då visas först Google Apps -inloggningsfönster och inbäddade filer visas efter inloggning.

Dokument

  1. Öppna Google Docs -dokument
  2. Klicka på Share och Publish to the Web (i övre högra hörn)
  3. Klicka på knappen Start publishing
  4. Kopiera Document link eller Embed code
  5. Lägg till i GoogleApps-block

Tabeller

  1. Öppna Google Docs -tabell
  2. Klicka på Share och Publish as a web page (i övre högra hörn)
  3. Klicka på knappen Start publishing
  4. Kopiera länken
  5. Lägg till i GoogleApps-block

Presentationer

  1. Öppna Google Docs -presentation
  2. Klicka på Share och Publish / embed (i övre högra hörn)
  3. Klicka på knappen Publish document
  4. Kopiera länken
  5. Lägg till i GoogleApps-block

Kollektioner

  1. Klicka på My collections eller Collections shared with me (listan till vänster) i Google Docs
  2. Klicka och välj kollektion som du vill bädda in
  3. Klicka på länken Sharing - Settings (till höger)
  4. Ändra tillgångsrättigheter till offentlig och spara förändringar
  5. Kopiera länken
  6. Lägg till i GoogleApps-block

Google Docs Viewer

Den här funktionen är likadan som förhandsvisning i Gmail. Du kan bädda in filer som blir direkt offentliga i internet. Kopiera filens hela adress (inklusive filnamn) och lägg till i GoogleApps-block.

Google Docs Viewer stöder följande filformat: PDF, TIFF, PPT, DOC, DOCX.

Obs: Filer som är uppladdade i Mahara är inte automatiskt offentliga och kan därför inte bäddas in.

Google Calendar

  1. Öppna Google Calendar
  2. Klicka på Settings och Calendar settings (menyn uppe till höger)
  3. Klicka på länken Calendars uppe till vänster
  4. Klicka på kalendernamn som skall bäddas in
  5. Det finns en funktion Embed This Calendar på mellanfliken Calendar details
  6. Redigera kalenderlayout och kopiera inbäddningskod
  7. Lägg till i GoogleApps-block

Google Maps

  1. Öppna Google Maps
  2. Klicka på länken My Maps uppe till vänster
  3. Klicka på kartans namn som skall bäddas in
  4. Klicka på länken Link uppe till höger
  5. Editera kartalayout och kopiera inbäddningskoden
  6. Lägg till i GoogleApps -block

Obs: Du kan passera punkt två och tre om du bäddar in andra än dina egna kartor.

Google Books

  1. Öppna Google Books
  2. Välj bok som du vill bädda in
  3. Klicka på länken eller knappen Preview this book
  4. Klicka på länken Link uppe till höger
  5. Kopiera bokens inbäddningskod
  6. Lägg till i GoogleApps-block
